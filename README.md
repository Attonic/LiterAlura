# LiterAlura

Projeto desenvolvido em Java para consumo de API de livros, com o objetivo de buscar, processar e exibir informações literárias de forma prática e automatizada.

##  Sobre o Projeto

O **LiterAlura** é uma aplicação focada no consumo da [API Gutendex](https://gutendex.com/), permitindo a pesquisa de obras e autores através de requisições web. O sistema processa os dados retornados e facilita a exploração de catálogos literários disponíveis em domínio público.

## Funcionalidades

- Busca de livros pelo título ou autor.
- Processamento e exibição de dados no formato JSON.
- Interação através de um menu na linha de comandos.

## Tecnologias Utilizadas

- **Java**
- **Spring Boot**
- **Maven** (Gerenciamento de dependências)
- **API Gutendex** (Fonte de dados literários)

## Como Executar o Projeto

Para correr a aplicação localmente, certifique-se de ter o Java instalado e siga os passos abaixo:

1. Clone este repositório:
   bash
   git clone https://github.com/Attonic/LiterAlura.git


2. Aceda à pasta do projeto:
   bash
   cd LiterAlura


3. Compile e execute o projeto utilizando o Maven Wrapper:
   bash
# Em Linux ou macOS
./mvnw spring-boot:run

# Em Windows
mvnw.cmd spring-boot:run
